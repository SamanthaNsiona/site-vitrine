# Projet Site Vitrine – Développement Web (HTML/CSS)

Ce projet a été réalisé dans le cadre du module de développement web, avec pour objectif de créer un **site vitrine professionnel** représentant une entreprise fictive. Le travail a été effectué en binôme, en respectant des critères techniques précis liés à la structure HTML, la présentation CSS et l’ergonomie générale du site.

---

## 1. Objectifs du projet

- Concevoir une interface web statique de type *site vitrine*.
- Respecter les bonnes pratiques de structuration HTML.
- Assurer une mise en forme adaptée via CSS.
- Appliquer des principes de lisibilité, hiérarchisation et cohérence visuelle.
- Reproduire une architecture professionnelle (header, navigation, sections, footer).

---

## 2. Technologies utilisées

| Domaine | Technologies |
|--------|--------------|
| Front-end | HTML5, CSS3 |
| Design web | Responsive design (flexbox / grid) |
| Collaboration | Travail en binôme – répartition des tâches et versioning local |

---

## 3. Fonctionnalités intégrées

- Page d’accueil avec présentation de l’entreprise.
- Section dédiée aux services ou produits.
- Page contact (ou intégration d’un formulaire statique).
- Mise en page structurée avec zones d’informations claires.
- Utilisation de typographies et styles cohérents avec l’identité visuelle choisie.

---

## 4. Organisation du projet

- Analyse des besoins selon le cahier des charges fourni.
- Conception de la structure HTML et mise en forme CSS.
- Travail en binôme : coordination et répartition des sections.
- Tests et ajustements visuels pour validation de la conformité.

---

## 5. Améliorations possibles

- Intégration d’animations ou transitions CSS avancées.
- Ajout d’interactivité (JavaScript).
- Optimisation des performances et de l’accessibilité (SEO & web performance).
- Adaptation du site en version mobile optimisée.
